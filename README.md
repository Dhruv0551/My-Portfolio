## 🚀 Getting Started

To run the portfolio locally:

```bash
# 1. Clone the repository
git clone https://github.com/Dhruv0551/my-portfolio.git

# 2. Navigate into the folder
cd my-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
